YAML是“另一种标记语言”的外语缩写，
YAML是"YAML Ain't a Markup Language"（YAML不是一种置标语言）的递归缩写。
在开发的这种语言时，YAML 的意思其实是："Yet Another Markup Language"（仍是一种置标语言），
