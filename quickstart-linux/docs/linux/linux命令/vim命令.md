vi显示行数

在vi的命令模式下，输入 :set nu
可以即时显示该文件的行数。
若每次使用vi时都需要显示行数，可以修改vi的配置文件，命令如下：
vi ~/.vimrc
进入该配置文件，输入 set nu
则每次使用vi都可显示行数
